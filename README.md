# Maison Fleur & Café

Responsive café website based on the Maison Fleur & Café Figma design.

## Stack

- React 19
- TypeScript
- Vite
- Responsive CSS design system
- Local Inter and Cormorant Garamond fonts
- GitHub Pages

## Features

- Responsive smartphone and desktop layouts
- Mobile navigation
- Expandable food menu
- Reservation dialog with confirmation state
- Dynamic Paris opening status
- Editorial image gallery using four custom café photographs

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

Pushes to `main` are built and published automatically through GitHub Pages.
